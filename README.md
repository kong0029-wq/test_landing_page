# Smart Planner Landing Page

> 복잡한 스케줄 관리는 AI에게 맡기고, 당신은 온전한 삶의 자유를 누리세요.

![Smart Planner](https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=1200&q=80)

## 📋 프로젝트 소개

Smart Planner는 AI 기반 시간 관리 서비스의 랜딩 페이지입니다. 현대인의 시간 빈곤 문제를 해결하고, 진정한 '여유'를 되찾아주는 서비스를 소개합니다.

## ✨ 주요 특징

- **프리미엄 디자인**: 다크/라이트 테마 교차, 자연 배경 이미지 활용
- **반응형 레이아웃**: 모바일, 태블릿, 데스크톱 완벽 지원
- **인터랙티브 요소**: 스크롤 애니메이션, 패럴랙스 효과, 탭 인터랙션
- **SEO 최적화**: 시맨틱 HTML5, 메타 태그 적용

## 🎨 디자인 시스템

| 요소 | 값 |
|------|-----|
| Primary Color | `#0A0A0A` (Dark) |
| Accent Color | `#E07A5F` (Coral) |
| Font (Title) | Playfair Display |
| Font (Body) | Noto Sans KR |

## 🛠 기술 스택

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (ES6+)
- Google Fonts
